NextJS 15 Multi-tenant e-commerce app.

NextJS, TypeScript, Tailwind CSS, tRPC, Payload CMS, Tanstack Query.

Payload CMS (with MongoDb) for backend, Stripe for payments.

## Getting Started

Development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
